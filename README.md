# ALPHA Shop

## 介紹

React開發，並以 CSS Modules 撰寫CSS樣式。
目前版本是將以下UI元素拆分成各元件，完成切版畫面。


### 完成元件

  - StepProgress
  - Step1-DeliverAddress
  - Step2-ShppingMethod
  - Step3-PaymentInformation
  - ProgressControl


## 安裝步驟
1. 透過終端機指令 git clone 複製專案至本地 
2. 在終端機輸入： npm start
3. 如果執行成功會進入到此網址 http://localhost:3000
4. 若欲暫停使用ctrl + c


## 開發工具
- React
- Visual Studio Code
- Node.js v14.18.1
